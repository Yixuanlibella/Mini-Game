Game Instructions

General Rules:

  • The game consists of 14 tiles displayed in a single row.
  • The tiles are divided into 5 groups (4 groups of 3 tiles and 1 group of 2 tiles).
  • Scoring Patterns:
    • Kezi: 3 identical tiles (points x 2).
    • Shunzi: 3 consecutive tiles of the same type (points x 2).
    • Duizi: 2 identical tiles in the last group (points x 2).
    • Hu: Achieved when all 4 groups are Kezi/Shunzi, and the last group is Duizi (total score doubled).
  • You can click on any tile to replace it with a random tile from the pool.
  • The game ends when the timer reaches 0.
